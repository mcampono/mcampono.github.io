A Tennessee Geographic Alliance educator workshop project using StoryMaps, web maps, and teacher-facing instructions to support humanities and geography education.
Published

June 1, 2022

The NEH Summer 2022 Workshop Geospatial Resources project was developed through volunteer work with the Tennessee Geographic Alliance to support teacher professional development, humanities education, geography instruction, and geospatial learning.

This project included a public workshop resource site, StoryMap instruction, web maps, and detailed teacher-facing instructional materials. The resources were designed to help educators use maps, spatial thinking, and geospatial storytelling to explore historical and geographic themes with students.

Project Role

My role included developing geospatial instructional materials, teaching teachers how to create StoryMaps, building web maps, writing teacher instructions, and supporting the use of geospatial tools in humanities-focused classroom activities.

Responsibilities included:

  • teaching educators how to use StoryMaps for classroom instruction,
  • creating web maps to support workshop themes and activities,
  • developing detailed teacher instructions,
  • building public-facing online workshop resources,
  • supporting teachers as they connected maps, geography, history, and storytelling,
  • creating examples that teachers could adapt for their own classrooms.

Instructional and Geospatial Goals

The project was designed to help educators:

  • use geospatial tools to support humanities instruction,
  • integrate maps and spatial thinking into historical inquiry,
  • create and use StoryMaps as instructional products,
  • explore migration, civil rights, place, movement, and landscape through maps,
  • connect geographic patterns to historical narratives,
  • develop classroom-ready activities using web maps and online resources.

Portfolio Significance

This project is a strong example of using geospatial technology to support humanities education. It demonstrates how StoryMaps, web maps, and teacher-facing instructional materials can help educators connect historical narratives with spatial evidence and geographic context.

It also reflects my broader outreach approach: geospatial tools are most effective when they are connected to meaningful questions, usable teaching materials, and clear guidance for educators.
